Detailed Description

Controller per la schermata principale dell'applicazione di gestione rubrica.

La classe gestisce la schermata principale, fornendo funzionalit√† come l'importazione, l'esportazione, l'aggiunta, la ricerca e l'ordinamento dei contatti. Implementa l'interfaccia Initializable per configurare l'interfaccia utente al momento dell'avvio.

La classe utilizza il pattern MVC per separare la logica della vista dalla logica del modello e comunica con il modello tramite il pacchetto Model.

Warning
L'applicazione non supporta file di rubrica in formati diversi dal CSV.
